# 对比与反思

  从计算结果的角度来看，两种方法较为相近。但是迭代的次数和运行的时间，高斯-赛德尔明显优于雅各比迭代法，收敛更快。但是应当注意，在某些情况下，雅各比可收敛，而高斯-赛德尔不会收敛，所以在使用过程中应当注意，并选择适当的方法。
